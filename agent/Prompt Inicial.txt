# 🗂️ Plan de Diseño — DataDay 2026 · Universidad del Caribe

## 🎨 Sistema de Diseño

**Paleta de colores extraída del flyer:**
| Token | Color | Hex |
|---|---|---|
| `--color-bg-deep` | Azul marino oscuro (fondo principal) | `#0A1628` |
| `--color-bg-mid` | Azul medio (hexágonos/gradiente) | `#0D2445` |
| `--color-accent-cyan` | Cian vibrante (DAY, íconos) | `#00E5FF` |
| `--color-accent-teal` | Teal suave (subtítulos) | `#00BCD4` |
| `--color-text-white` | Blanco (DATA, cuerpo) | `#FFFFFF` |
| `--color-hex-gray` | Gris azulado (hexágonos fondo) | `#1E3A5F` |

**Tipografía (inspirada en el flyer):**
- **Orbitron** → Titulares y el nombre DATA DAY (Google Fonts, gratuita, es la que más se acerca al look del flyer)
- **Exo 2** → Cuerpo de texto, descripciones, horarios (legible, tech, complementa bien)

## 🏗️ Arquitectura de la Página

```
/                    → Inicio (Hero + info general)
/#horario            → Horario del día
/#conferencias       → Conferencias / Ponencias
/#talleres           → Talleres
/#feria              → Feria de Proyectos
/#comunidades        → Comunidades invitadas
/#patrocinadores     → Patrocinadores
```
Una sola página con scroll (SPA), navegación sticky tipo navbar que resalta la sección activa conforme bajas.

---

## 📐 Secciones — Descripción detallada

### 1. `NAVBAR` — Fija en scroll
- Logo DataDay 2026 a la izquierda (pequeño, solo texto estilizado)
- Links a cada sección a la derecha
- Fondo translúcido con blur al hacer scroll (`backdrop-filter: blur`)
- Botón CTA tipo "Regístrate" en cian si hay registro, si no → "Ver Programa"

---

### 2. `HERO` — Sección de inicio (pantalla completa)
- **Fondo:** Gradiente del azul marino + patrón de hexágonos animado (CSS o SVG, igual que el flyer)
- **Centro:** Logo grande del DataDay 2026 (usa la imagen que me mandaste)
- **Subtítulo:** "Conferencias · Talleres · Networking"
- **Chips de info:** 📅 Jueves 29 de abril · ⏰ 10:00–18:00 hrs · 📍 Auditorio Ingenierías
- **CTA Button:** "Ver Programa" → scroll suave a #horario
- **Carrusel debajo del hero** (o integrado): slides de flyers de conferencias/talleres, auto-play con transición fade

---

### 3. `SOBRE EL EVENTO` — Strip de 3 columnas
Tres tarjetas con ícono cian, título y descripción breve:
- 🎤 **Conferencias** — Expertos de la industria y academia
- 🛠️ **Talleres** — Aprende haciendo con sesiones prácticas
- 🤝 **Networking** — Conecta con profesionales y comunidades tech

---

### 4. `HORARIO` — Timeline visual
- Diseño de línea de tiempo vertical (o tabla de dos columnas mañana/tarde)
- Cada bloque: hora · nombre de actividad · sala/espacio · tipo (🎤 conferencia / 🛠️ taller / ⭐ especial)
- Colores distintos por tipo de actividad usando variantes del cian y teal
- Sticky de "hora actual" si el evento está en curso (bonus)

---

### 5. `CONFERENCIAS & PONENCIAS` — Grid de cards
- Cards con: foto del ponente, nombre, cargo/empresa, título de la charla, horario
- Hover: el card se eleva con glow cian
- Si no hay foto disponible, avatar placeholder con inicial sobre fondo hexagonal

---

### 6. `TALLERES` — Grid similar pero con detalle extra
- Igual que conferencias pero con: cupo máximo, si requiere laptop, nivel (básico/intermedio/avanzado)
- Badge de "Cupo limitado" en rojo/naranja si aplica

---

### 7. `FERIA DE PROYECTOS` — Sección tipo "spotlight"
- Layout de ancho completo con fondo ligeramente diferente (hex pattern más denso)
- Texto explicativo de cómo funcionará: stands, qué se exhibe, cómo participar
- Lugar para poner el flyer específico de la feria como imagen destacada
- Botón "Conoce más" / "Registra tu proyecto" si aplica

---

### 8. `COMUNIDADES` — Cards con logo + descripción
- Grid de 2–4 cards (escalable)
- Cada card: logo de la comunidad, nombre, descripción de 1 línea
- Estilo glassmorphism sobre fondo oscuro
- Si hay pocas comunidades (1–2), se puede hacer un layout más generoso tipo "featured partners"

---

### 9. `PATROCINADORES` — Logo wall
- Tres niveles si aplica: Oro / Plata / Bronce (o simplemente una fila de logos)
- Logos en blanco/monocromático sobre fondo oscuro para consistencia visual
- Título: "Gracias a nuestros patrocinadores"

---

### 10. `FOOTER`
- Logo Universidad del Caribe + DataDay 2026
- Links rápidos a secciones
- Redes sociales (íconos cian)
- "Universidad del Caribe · Cancún, Quintana Roo, México"

## 🔧 Stack Tecnológico para Google IDX / Firebase

Como mencionas **Google IDX** (antigravity = probablemente te refieres a esto), te recomiendo:

| Capa | Tecnología |
|---|---|
| Framework | **Astro** o **HTML/CSS/JS vanilla** (más simple para un evento estático) |
| Estilos | **CSS custom properties** (variables de color arriba) + Flexbox/Grid |
| Fuentes | Google Fonts: `Orbitron` + `Exo 2` |
| Animaciones | CSS keyframes + `Intersection Observer` para animar al scroll |
| Hosting | **Firebase Hosting** (gratis, rápido, ideal para este caso) |
| Imágenes | Firebase Storage o simplemente carpeta `/assets` |

## 📝 Prompt sugerido para construir la página en IDX

```
Build a single-page event website for "DataDay 2026" at Universidad del Caribe, Cancún.

**Tech stack:** HTML, CSS (custom properties, no framework), vanilla JS.

**Design system:**
- Colors: --bg-deep: #0A1628; --bg-mid: #0D2445; --accent-cyan: #00E5FF; 
  --accent-teal: #00BCD4; --text: #FFFFFF; --hex-gray: #1E3A5F
- Fonts: 'Orbitron' (headings), 'Exo 2' (body) — import from Google Fonts
- Background motif: repeating SVG hexagon pattern (like the flyer)

**Sections (single scroll page with smooth anchor nav):**
1. Sticky navbar with logo + links + CTA button
2. Hero: full-screen with animated hex background, DataDay logo image, 
   event chips (date: April 29 / time: 10:00–18:00 / venue: Auditorio Ingenierías), 
   and image carousel below for conference/workshop flyers
3. About: 3-column feature strip (Conferencias, Talleres, Networking)
4. Schedule (#horario): vertical timeline with color-coded activity types
5. Conferences (#conferencias): speaker cards grid with glow hover effect
6. Workshops (#talleres): cards with difficulty badge and requirements
7. Project Fair (#feria): full-width spotlight section with image placeholder
8. Communities (#comunidades): glassmorphism logo cards (2–4 items)
9. Sponsors (#patrocinadores): logo wall with tier labels
10. Footer with university branding and social links

**Notes:**
- All text in Spanish
- Use CSS animations on scroll (Intersection Observer)
- Mobile responsive (hamburger menu on mobile)
- Cyan glow effects on interactive elements
- University logo in navbar and footer
```
